### 📖 Inti Materi (Cobra Course)
1. Apa itu Looping?
Looping adalah instruksi untuk menjalankan blok kode yang sama secara berulang selama kondisi tertentu terpenuhi. Di episode ini, kita fokus pada FOR Loop.
2. Anatomi FOR Loop
   ```bash
   for (int i = 1; i <= 10; i++) 
    {
        // Kode di sini akan diulang 10 kali
    }
   ```
   * Inisialisasi (int i = 1): Titik awal perulangan.
   * Kondisi (i <= 10): Selama ini benar (True), perulangan lanjut.
   * Iterasi (i++): Apa yang terjadi setelah satu putaran selesai? (Dalam hal ini, tambah 1).
3. Kenapa Harus Pakai Loop?
   Daripada menulis Console.WriteLine sebanyak 100 kali, kita cukup menulisnya 1 kali di dalam loop. Ini membuat kode kita DRY (Don't Repeat Yourself).

---
Di episode ini, kita mempelajari cara memberikan "otak" pada program agar bisa mengambil keputusan berdasarkan kondisi tertentu.
📖 Inti Materi (Cobra Course)
### 1. Apa itu IF/ELSE?

Logika percabangan adalah cara kita mengatur alur program. Program akan mengecek sebuah Kondisi (benar atau salah):

    IF (Jika): Bagian kode yang jalan hanya jika kondisinya BENAR.

    ELSE (Selain itu): Bagian kode yang jalan jika kondisinya SALAH.

### 2. Operator Perbandingan

Untuk membuat keputusan, kita menggunakan operator:

    > (Lebih besar)

    < (Lebih kecil)

    >= (Lebih besar atau sama dengan)

    <= (Lebih kecil atau sama dengan)

    == (Sama dengan)

    != (Tidak sama dengan)

### 3. Contoh Kasus

Jika kita ingin membatasi akses berdasarkan umur:
C#

```bash
// Syarat: Minimal 18 tahun
if (umur >= 18) {
    // Jalur ini diambil jika umur 18, 19, 20...
} else {
    // Jalur ini diambil jika umur 17, 16, 15...
}
```
